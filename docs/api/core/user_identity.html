class UserIdentityManager (conversation_memory=None)

Manages user identity for conversational recipe management.

Supports multiple identification modes: 1. Session-based: Temporary user ID for current session only 2. Named: User-provided name/identifier for persistence 3. Environment: User ID from environment variable for development

Initialize user identity manager.

Args

conversation_memory
Optional ConversationMemory instance for integration
