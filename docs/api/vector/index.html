Module src.vector

Vector database operations for semantic recipe search and RAG.

Sub-modules

src.vector.embeddings

Recipe embedding generation using OpenAI's text-embedding-ada-002 model. Handles text preparation and embedding creation for vector search.

src.vector.filters

Recipe filtering utilities for vector search operations.

src.vector.ingestion

Recipe ingestion pipeline for loading recipes into vector database. Handles batch processing and data migration from example recipes.

src.vector.keywords

Keyword extraction utilities for sparse search functionality. Provides text preprocessing and tokenization for BM25 search.

src.vector.store

Vector database operations for recipe storage and retrieval. Provides a high-level interface to Chroma DB operations.

src.vector.types

Type definitions for vector database operations. Centralized location for all vector-related type definitions.

src.vector.user_collections

User recipe collections management for the vector database. Provides functionality for user-specific recipe storage, validation, and retrieval.

Get or create the Chroma database client connection.

Lazily initializes the HTTP client connection to ChromaDB and tests connectivity with a heartbeat. Connection parameters are loaded from the vector configuration.

Get or create the recipe collection in ChromaDB.

Lazily initializes the collection, attempting to get an existing collection first, then creating a new one if it doesn't exist.
